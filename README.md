# 📊 Super Store Sales Dashboard

## 📌 Overview
This project is a Power BI dashboard built using Super Store sales data to analyze business performance.

## 🔍 Key Insights
- Total Sales: 1.57M
- Total Quantity: 22K
- Total Profit: 175K
- Average Delivery Time: 23K

## 📊 Dashboard Features
- Sales by Category & Sub-Category
- Sales by Region & State (Map Visualization)
- Sales by Segment (Consumer, Corporate, Home Office)
- Sales by Payment Mode
- Sales by Ship Mode
- Monthly Sales Trend

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset
